# Employee Payroll Management System
Mini Project for DBMS Course  
Created by: Disha Andre, Anishka Sah, Ankitkumar Shah

## Description
This project involves designing and implementing a Payroll Management System using MySQL. 
The system handles employee records, department details, salary processing, and attendance tracking.

## Files Included
- `Payroll_SQL_Queries.pdf` – SQL queries for schema & data
- `Payroll_Presentation.pdf` – PowerPoint presentation

## Features
- Normalized DB schema (1NF, 2NF, 3NF)
- SQL implementation with constraints
- Realistic payroll & attendance management

## Technologies
- MySQL
- ER Modeling
